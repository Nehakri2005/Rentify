#  Rentify – Property Rental Platform

##  Overview  
**Rentify** is a full-stack web application that allows users to **rent properties or list their own properties for rent**. The platform simplifies the renting process by providing features like property listing, searching, filtering, reviews, and map-based location tracking.  

---

## 🚀 Features  

### 👤 User Authentication  
- Secure **Signup & Login**  
- User session management  

###  Property Management  
- Add new properties  
- Edit property details  
- Delete property listings  
- Upload images and details  

### 🔍 Property Exploration  
- Browse all available properties  
- Advanced **filtering options** (trending, islands, etc.)  
- Search functionality  

### 🗺️ Map Integration  
- View property location on map  
- Better understanding of nearby areas  

### ⭐ Reviews & Ratings  
- Users can **add reviews** for properties  
- Helps others make better decisions  

### 📱 User Experience  
- Clean and responsive UI  
- Easy navigation for both renters and owners
